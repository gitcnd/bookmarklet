## Troubleshooting

**Nothing happens when I click**
- Make sure you're on a supported site
- Check that you copied the entire bookmarklet code
- Try refreshing the page and clicking again

**File is empty**
- The page might not have loaded completely
- Try scrolling through the conversation first
- Some very new conversations might not export properly

**Wrong site detected**
- Make sure you're on the correct domain
- Clear your browser cache
- Try creating the bookmark again

## Tips

- Export conversations regularly to avoid losing them
- Files are named based on the conversation title
- All exports use the same Markdown format
- You can edit the downloaded files in any text editor
